# Source Tree  설치

1. Create [Github](https://github.com/) account

2. Ask Git manager for the access to GE Appliances repository

3. Generate Personal Access Token
    1. Click on profile icon on the top right corner in Github 
    2. Select Settings 
      
    <img align="left" src="./imgs/3-2.png" width="200" /><br/>
    
    2. Select Developer Settings

    <img align="left" src="./imgs/3-3.png" width="200"  /><br/>

    3. Select Personal access tokens

    4. Click on Generate new token button

    <img align="left" src="./imgs/3-4,5.png" width="700"  /><br/>

    5. Enter Note for the name of token, and select all scopes for the full access
    <img align="left" src="./imgs/3-6.png" width="600"  /><br/>
    <img align="left" src="./imgs/3-6-1.png" width="600"  /><br/>

    6. Click on Generate token button

    7. Keep generated token safe for later

    8. Authorize the generated token to enable SSO

    <img align="left" src="./imgs/3-8.png" width="500"  /><br/>

4. Download and install [SourceTree](https://www.sourcetreeapp.com/)

5. Set up Github account for SourceTree
    1. Select File > Settings

    2. Select Account tab

    3. Click Add button and set specifics as below

    <img align="left" src="./imgs/5-3.png" width="500"  /><br/>

    4. Clone private repository to confirm the connection

    <img align="left" src="./imgs/5-4.png" width="500"  /><br/>
